🔄 Atualizar grupos (Admin)
1
No computador, abra o Clik Vendas → selecione o plano → clique no ícone MDC → ⚡ Importar todos os grupos
2
Clique 💾 Baixar JSON — arquivo mdc_grupos_XX.json é baixado
3
Acesse app.netlify.com → seu site → Deploys → arraste o arquivo JSON renomeado para grupos.json
4
Pronto! Todos os vendedores veem os grupos atualizados ao clicar 🔄 na aba Grupos
📱 Instalar no celular como App
🤖
Android: Abra no Chrome → menu ⋮ → "Adicionar à tela inicial"
🍎
iPhone: Abra no Safari → botão compartilhar □↑ → "Adicionar à Tela de Início"
